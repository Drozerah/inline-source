# Inline Source

> Inline all `<script>`, `<link>` and `<img>` tags that contain the inline attribute with

### Installation

````bash
$ npm install
````

### Usage

Create ./dist output with inline content...
````bash
$ npm run build
````
Clean ./dist
````bash
$ npm run clean:build
````

### NPM Script details

````json
{
  "scripts": {
    "clean:build": "rm -rf ./dist",
    "build": "gulp inlinesource"
  },
}
````
